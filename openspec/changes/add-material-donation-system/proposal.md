# 变更：添加物资捐赠与追踪系统

## 为什么

当前系统仅支持资金捐赠，无法满足捐赠者提供实物物资的需求。同时，帮扶对象的选择缺乏科学推荐机制，捐赠者和公众无法实时了解物资流向和帮扶进度，影响了平台的透明度和公信力。

## 变更内容

### 1. 帮扶对象推荐功能
- **智能推荐引擎**：根据儿童需求、紧急程度、地区等因素推荐帮扶对象
- **多维度筛选**：按年龄段、地区、需求类型（教育/生活/心理）筛选
- **推荐理由展示**：说明为什么推荐该对象，增加信任度
- **个性化推荐**：根据捐赠者偏好（地区、年龄段）定制推荐

### 2. 物资捐赠功能
- **物资分类管理**：衣物、书籍、学习用品、生活用品、体育用品等
- **物资信息登记**：名称、数量、新旧程度、价值估算、图片上传
- **捐赠方式选择**：邮寄到指定地点 / 上门取件（限特定城市）
- **物资需求匹配**：根据受助儿童需求，展示急需物资清单
- **捐赠确认**：接收确认、质量检查、入库登记

### 3. 物资流向追踪
- **全流程记录**：捐赠 → 接收 → 检验 → 入库 → 分配 → 发放 → 签收
- **每个节点拍照存证**：关键环节上传照片证明
- **物流信息集成**：对接快递公司 API，实时追踪物流状态
- **时间线展示**：捐赠者可查看物资流转的完整时间线
- **签收反馈**：受助儿童签收照片/视频反馈

### 4. 帮扶进度信息公开
- **进度可视化**：时间轴、进度条、地图标注等方式展示
- **定期更新**：志愿者定期上传帮扶活动照片、文字记录
- **效果评估**：定期发布帮扶效果报告（学习成绩改善、生活条件提升）
- **数据统计**：物资发放统计、受益人数、覆盖地区
- **透明公示**：所有信息对捐赠者和公众公开

## 影响

### 受影响的规范
- **新增规范**：
  - `material-donation` - 物资捐赠管理
  - `recommendation` - 帮扶对象推荐
  - `tracking` - 物资流向追踪
  - `progress-disclosure` - 帮扶进度公开

### 受影响的代码模块
- **后端**：
  - 新增 MaterialDonationController（物资捐赠接口）
  - 新增 RecommendationController（推荐引擎接口）
  - 新增 TrackingController（追踪接口）
  - 新加 ProgressController（进度公开接口）
  - 扩展 Child 实体（增加推荐相关字段）

- **前端网站**：
  - 新增物资捐赠页面
  - 新增帮扶对象推荐页面
  - 新增物资追踪页面
  - 新增帮扶进度展示页面

- **管理后台**：
  - 新增物资管理模块
  - 新增推荐配置模块
  - 新增流向追踪管理
  - 新增进度发布模块

### 数据库变更
需要创建以下新表：
- `material_donation` - 物资捐赠记录
- `material_item` - 物资明细
- `material_demand` - 物资需求
- `donation_tracking` - 捐赠追踪记录
- `recommendation_log` - 推荐日志
- `help_progress` - 帮扶进度记录
- `progress_attachment` - 进度附件（照片/视频）

### 非功能性影响
- **性能**：推荐算法需要快速响应（< 500ms）
- **存储**：物资追踪照片和视频需要额外的存储空间
- **安全**：捐赠者隐私保护，受助儿童信息脱敏
