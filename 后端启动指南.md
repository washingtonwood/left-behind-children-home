## 🚀 启动步骤

### 方法一：使用启动脚本（推荐）

1. **双击运行** `启动后端.bat`
2. **等待编译完成**（首次启动需要2-3分钟）
3. **看到以下提示表示成功**：
   ```
   Started ShouhutongxingApplication in xx seconds
   ```
4. **后端服务运行在**：http://localhost:8080/api

### 方法二：手动启动

1. **打开命令提示符（CMD）**
2. **切换到后端目录**：
   ```cmd
   cd D:\bi\shouhutongxing-backend
   ```
3. **设置Java环境**：
   ```cmd
   set JAVA_HOME=C:\Program Files\Eclipse Adoptium\jdk-25.0.1.8-hotspot
   set PATH=%JAVA_HOME%\bin;%PATH%
   ```
4. **启动服务**：
   ```cmd
   mvnw.cmd spring-boot:run
   ```

## ⚠️ 常见问题

### 问题1：端口8080被占用
**解决方案**：
```cmd
# 查找占用端口的进程
netstat -ano | findstr :8080

# 结束进程（替换PID）
taskkill /F /PID 进程ID
```

### 问题2：MySQL连接失败
**解决方案**：
```cmd
# 检查MySQL服务
netstat -ano | findstr :3306

# 启动MySQL服务
net start MySQL
```

### 问题3：编译失败
**解决方案**：
1. 确认Java版本 ≥ 17
2. 清理Maven缓存：
   ```cmd
   mvnw.cmd clean
   ```
3. 重新下载依赖：
   ```cmd
   mvnw.cmd dependency:resolve
   ```

### 问题4：前端无法连接后端
**检查清单**：
- ✅ 后端已启动（端口8080）
- ✅ 浏览器控制台没有CORS错误
- ✅ API地址正确（http://localhost:8080/api）
- ✅ 后端控制台没有报错
